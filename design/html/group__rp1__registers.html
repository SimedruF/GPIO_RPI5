Atomic Access Offsets

Added to the RIO base to perform atomic bit operations without read-modify-write.

#define RP1_CLR_OFFSET   0x3000

RIO Register Offsets

Offsets within each SYS_RIOx block for output, output-enable, and input.

#define RIO_OUT   0x00
 Output value register.
 
#define RIO_OE   0x04
 Output enable register (1 = output, 0 = input).
 
#define RIO_IN   0x08
 Input value register (read-only).
