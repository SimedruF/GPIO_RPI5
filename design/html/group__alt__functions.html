Detailed Description

FUNCSEL values for the RP1 GPIO controller.

Each GPIO pin can be muxed to one of several peripheral functions (SPI, I2C, UART, PWM, PCM, etc.) by writing the corresponding FUNCSEL value to the pin's control register. The mapping varies per pin — consult the RP1 peripherals datasheet for the full pin-function matrix.
